**Running Apache Spark**
in terminal:
- docker pull bitnami/spark
- docker run 

**Running Apache Hadoop**
in terminal (worker node):
 - docker pull bde2020/hadoop-datanode
 - docker run bde2020/hadoop-datanode

in terminal (master node):
 - docker pull bde2020/hadoop-namenode
 - docker run bde2020/hadoop-namenode

**Running SonarQube/SonarScanner**
in terminal:
- docker pull sonarqube
- docker run sonarqube

**Running Jupyter Notebook**
in terminal:
- docker pull jupyter/minimal-notebook
- docker run jupyter/minimal-notebook
